Better Bogleheads improves your browsing experience on the Bogleheads.org home page.

FEATURES

Zebra Striping
Add alternating row colors to the thread list for easier reading. Choose any color you like.

Hide Read Threads
Automatically tracks threads you've clicked and lets you hide them. Threads reappear when new replies are posted, so you never miss new activity.

Unread Badge
Shows the count of unread threads on the extension icon. The badge color changes based on how many unread threads you have: green (1-20), yellow (21-60), or red (61+).

Highlight Hot Topics
Make popular threads stand out by highlighting threads above a customizable reply threshold. Choose your own highlight color.

Hide Old Threads
Filter out threads older than a specified number of days to focus on recent discussions.

Adjustable Font Size
Increase or decrease the thread list font size from 50% to 200% for comfortable reading.

Pointer Cursor
Enable a pointer cursor when hovering over thread rows to make them feel more clickable.

Time Tracking
See how much time you spend on Bogleheads.org. The side panel displays your total time, days tracked, and average time per day. Reset anytime to start fresh.

Omnibox Commands
Type "bh" in Chrome's address bar, press Tab, then use these shortcuts:
• new - Go to active topics
• hot / hot off - Toggle hot topic highlighting
• zebra / zebra off - Toggle zebra striping
• font inc / font dec - Adjust font size
• reset - Clear read history
• #12345 - Go to thread by ID
• user:alice - Search posts by user
• "search term" - Search the forum

Collapsible Settings Panel
The settings panel floats in the top-right corner and can be collapsed to stay out of your way.

Draggable Settings Panel
Drag the panel by its header to reposition it anywhere on screen. Your preferred position is saved automatically.

All settings are saved automatically and persist across sessions.

PRIVACY

This extension only runs on https://bogleheads.org/ and stores your preferences locally using Chrome's storage API. No data is collected or transmitted.

OPEN SOURCE

This extension is open source and available on GitHub. Contributions and feedback are welcome.
